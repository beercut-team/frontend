# PEAK-IT 2026

Конкурсный проект для **PEAK-IT 2026**.

## Команда

| Роль | Участник |
|------|----------|
| Менеджер | Свинобоев Артем |
| Дизайнер | Попов Айтал |
| Разработчик | Чичигинаров Остап |

## Стек

- **SvelteKit 2** + **Svelte 5**
- **TypeScript**
- **Tailwind CSS v4**
- **Feature-Sliced Design (FSD)**
- **bun** — пакетный менеджер

## Запуск

```bash
bun install
bun run dev
```

## Сборка

```bash
bun run build
bun run preview
```
